## 📖 About the Project

The **Amazon Sales Analytics Dashboard** is a comprehensive Power BI solution designed to surface actionable insights from Amazon's sales data. It consolidates performance metrics across multiple business dimensions into a single, interactive interface — eliminating data silos and enabling faster, more informed decision-making.

---

## 📊 Dashboard Features

The dashboard covers several core areas of analysis:

| Area | Description |
|------|-------------|
| 💰 **Overall Sales Performance** | Tracks total revenue, costs, and profit margins to reflect the overall financial health of the business |
| 🛒 **Sales Channel Analysis** | Breaks down how online and offline channels each contribute to total sales via pie and bar charts |
| 🚦 **Order Priority Analysis** | Examines how different order priority levels (Critical, High, Medium, Low) influence revenue and profitability |
| 🌍 **Geographic Distribution** | Visualizes sales across global regions using an interactive world map to identify key markets |
| 📦 **Product Performance** | Surfaces the most and least profitable item categories through revenue and profit comparisons |
| 📈 **Time-Series Trends** | Tracks revenue and profit over time to reveal seasonal patterns, growth trajectories, and anomalies |

---

## ⚙️ Key Capabilities

- 🔍 **Interactive Filters** — Slice data by date range, sales channel, order priority, and region
- 🔎 **Drill-Down Support** — Navigate from high-level summaries down to granular transaction-level details
- 🎨 **Customizable Layouts** — Adjust visualizations and dashboard arrangements to match reporting needs
- 📤 **Export Ready** — Share insights with stakeholders via Power BI's built-in export and sharing features

## 🎯 Project Goals

- ✅ Provide a clear, concise overview of Amazon's sales performance
- ✅ Identify key trends and patterns hidden within large volumes of sales data
- ✅ Support data-driven decision-making with actionable, real-time insights
- ✅ Foster collaboration by giving all stakeholders a shared, reliable source of truth
